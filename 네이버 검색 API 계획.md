# 🚀 TrendRadar v3.0: 데이터 기반 블루오션 SEO 파이프라인 개선안

## 1. 개요 (Overview)
본 문서는 신생 블로그의 검색 시장 생존율을 극대화하기 위한 **TrendRadar 시스템의 3세대(v3.0) 아키텍처** 전환 계획을 담고 있습니다. 기존의 'AI 추론형 기획'에서 발생하는 불확실성을 제거하고, **실제 검색 데이터(Data-Driven)**를 파이프라인의 핵심 지표로 삼아 '무조건 노출되는 콘텐츠'를 생산하는 것을 목적으로 합니다.

---

## 2. 현행 방식의 한계 및 개선 필요성
* **AI의 정보 공백:** 실시간 검색 기능이 제한된 환경에서 AI는 키워드의 실제 경쟁 강도를 '짐작'할 뿐이며, 이는 레드오션 키워드 발행으로 인한 리소스 낭비로 이어집니다.
* **데이터 신뢰도 문제:** 기획 단계에서 수치나 통계가 부족할 경우 모델이 허구의 팩트를 생성(Hallucination)할 위험이 존재합니다.
* **신생 블로그의 낮은 지수:** 도메인 점수가 낮은 신생 블로그는 경쟁 지수가 조금이라도 높은 키워드에서는 대형 블로그에 밀려 노출 기회를 얻기 어렵습니다.

---

## 3. 핵심 솔루션: 3-Step 전략적 파이프라인
기존 2단계(기획-본문) 공정을 3단계로 세분화하고, 가성비 모델과 고성능 모델을 적재적소에 배치합니다.

### 1단계: 지능형 키워드 정제 (Keyword Extraction)
* **담당 모델:** `Gemini Flash Light` (속도 및 비용 최적화)
* **수행 내용:** 뉴스 헤드라인이나 커뮤니티 제목 등 노이즈가 많은 원문 데이터에서 네이버 API 조회에 적합한 **핵심 검색어(Search Term)**만 정제하여 추출합니다.
* **기대 효과:** 문장형 데이터를 분석 가능한 단어 단위로 변환하여 시스템의 전반적인 정확도를 높입니다.

### 2단계: 실시간 데이터 검증 (Data Verification)
* **연동 도구:** 네이버 검색광고 API (조회수) + 네이버 블로그 검색 API (문서 수)
* **수행 내용:** 추출된 키워드를 서버단에서 즉시 타격하여 다음 지표를 산출합니다.
    * **월간 검색 볼륨:** 실제 유입 가능한 트래픽의 크기 확인.
    * **경쟁 지수 (Competition Index):** `발행 문서 수 / 검색량` 연산을 통해 블루오션 여부 판단.
* **필터링:** 검색량이 기준치 미만이거나 경쟁 지수가 높은 '레드오션' 기획안은 이 단계에서 가차 없이 폐기(Drop)합니다.

### 3단계: 팩트 기반 전략 기획 (Strategy Planning)
* **담당 모델:** `Gemini Flash` (논리 및 창의성 집중)
* **수행 내용:** 검증을 통과한 최정예 키워드와 실제 수치 데이터를 바탕으로 최종 기획안을 도출합니다.
* **기대 효과:** AI가 "이 키워드는 경쟁률이 0.5 미만인 블루오션이다"라는 팩트를 인지한 상태에서 가장 날카로운 제목과 정보 격차(`serpDifferentiation`)를 설계합니다.

---

## 4. 기술적 구현 및 최적화 포인트
* **병렬 처리 아키텍처:** `Promise.all`을 사용하여 다수의 키워드에 대한 API 요청을 동시 처리함으로써, 파이프라인 지연 시간을 2초 이내로 유지합니다.
* **보안 통신:** 네이버 검색광고 API의 까다로운 HMAC-SHA256 서명 규격을 서버단에서 완벽히 구현하여 안정적인 데이터 수급을 보장합니다.
* **프롬프트 고도화:** 기획 프롬프트에 네이버 지표 변수(`{{searchVolume}}`, `{{competitionIndex}}`)를 주입하여 AI가 실제 데이터를 본문에 자연스럽게 녹여내도록 유도합니다.

---

## 5. 기대 효과 (Expected Outcomes)
1. **발행 효율 200% 상승:** '유입 0명'인 포스팅을 원천 차단하여 시스템 운영 비용 대비 트래픽 효율을 극대화합니다.
2. **SEO 신뢰도 점수 확보:** 구체적인 수치와 팩트가 담긴 글은 검색 엔진(SGE 등)에 의해 고품질 문서로 분류될 확률이 높습니다.
3. **완전 자동화 수익화:** 꿀 키워드 발굴부터 발행까지 사람의 개입 없이 데이터 기반으로 작동하는 강력한 현금 흐름 파이프라인을 구축합니다.
